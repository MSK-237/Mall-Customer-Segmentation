# Mall Customer Segmentation

This project performs customer segmentation using KMeans Clustering based on annual income and spending score.

## Project Features
- Data preprocessing
- Data visualization
- Feature scaling
- KMeans clustering
- Elbow Method
- Silhouette Score
- Customer segmentation
- Streamlit deployment
- Hugging Face deployment

## Technologies
- Python
- Pandas
- Scikit-learn
- Matplotlib
- Seaborn
- Streamlit

## Clustering Performance

| Metric | Score |
|---|---|
| Silhouette Score | BURAYA-SCORE |
